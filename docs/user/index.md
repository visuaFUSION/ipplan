## Quick Reference

### Common Tasks

| Task | Location |
|------|----------|
| View subnets | Network > Subnets > Display Subnet Information |
| Search for IP | Network > Subnets > Search All Subnets |
| Find free space | Network > Subnets > Find Free Space |
| Change password | Options > Change my Password |

### Navigation

IPplan uses a menu system at the top of the page:

- **Main** - Return to home page
- **Customers** - Manage customers (if permitted)
- **Network** - IP address management
- **DNS** - DNS records (if enabled)
- **Options** - Your settings
- **Help** - Documentation
- **Logout** - End your session
